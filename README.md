# 🎨 Launcher Assets Storage
Репозиторий для хранения графических ресурсов и статических файлов моего лаунчера Minecraft на Delphi.
📦 Что здесь лежит:
Backgrounds — фоновые изображения для форм (UrlBitmapForm и другие).
Icons — иконки кнопок, модов и версий.
UI Elements — элементы интерфейса, кастомные прогресс-бары и логотипы.
JSON Configs — (опционально) списки версий или новости.
🔗 Как использовать в Delphi
Все файлы доступны по прямым ссылкам через GitHub Raw или другие привязанные сервисы.
Пример получения картинки:
pascal
const
  UrlBackground = 'https://raw.githubusercontent.com';
Используйте код с осторожностью.

🛠 Обновление ресурсов
Чтобы обновить картинку в лаунчере у всех пользователей сразу:
Замени файл в этом репозитории (сохранив то же имя).
Сделай Commit и Push.
Лаунчер при следующем запуске подтянет обновленный файл по ссылке.
