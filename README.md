# 🖼️ My Launcher Assets

Склад всех картинок и ресурсов для моего лаунчера Minecraft на Delphi.

### 📂 Содержимое
* **Backgrounds** — Обои и фоны для главного меню.
* **Buttons** — Текстуры кнопок и элементов интерфейса.
* **Icons** — Иконки лаунчера.

### 🔗 Прямые ссылки
Используй `Raw` ссылки из этого репозитория в коде на Delphi через `TIdHTTP`, чтобы подгружать графику динамически.
